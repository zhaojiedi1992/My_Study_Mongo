3. 数据增删¶

3.1. 数据库创建¶

> use mongo
switched to db mongo
> db
mongo
> show dbs
admin  (empty)
local  0.078GB
> db.mongo.insert({"name":"zhojiedi","age":"26"})
WriteResult({ "nInserted" : 1 })
> show dbs
admin  (empty)
local  0.078GB
mongo  0.078GB

3.2. 删除数据库¶

> use mongo
switched to db mongo
> db.dropDatabase()
{ "dropped" : "mongo", "ok" : 1 }
> show dbs
admin  (empty)
local  0.078GB

3.3. 删除集合¶

db.collection.drop()
